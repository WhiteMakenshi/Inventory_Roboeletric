# Inventory App

Inventory App es una aplicación de Android desarrollada en Kotlin que te permite gestionar un inventario de productos. La aplicación cuenta con las siguientes características:

*   **Visualización de productos:** Muestra un listado de todos los productos en una cuadrícula personalizable.
*   **Añadir productos:** Permite añadir nuevos productos al inventario a través de un formulario.
*   **Eliminar productos:** Ofrece la posibilidad de eliminar productos existentes del inventario.
*   **Arquitectura MVVM:** La aplicación sigue el patrón de arquitectura Model-View-ViewModel (MVVM) para una separación clara de las responsabilidades y un código más mantenible.
*   **Componentes de Android Jetpack:** Utiliza componentes modernos de Android Jetpack como ViewModel y LiveData para la gestión de datos y la actualización de la interfaz de usuario.
